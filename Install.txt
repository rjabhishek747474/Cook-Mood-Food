# DailyCook Installation Guide

## Prerequisites
- Python 3.10+
- Node.js 18+ & npm
- Git
- Docker (optional)

## Quick Start (Docker)
1. git clone https://github.com/rjabhishek747474/Cook-Mood-Food
2. cd Cook-Mood-Food
3. docker-compose up -d
4. docker-compose exec backend python -m scripts.seed_data
5. Open http://localhost:3000

## Manual Setup
### Backend
1. cd backend
2. python -m venv venv
3. .\venv\Scripts\activate (Windows) or source venv/bin/activate (Mac/Linux)
4. pip install -r requirements.txt
5. python -m scripts.seed_data
6. uvicorn main:app --reload

### Frontend
1. cd frontend
2. npm install
3. npm run dev
4. Open http://localhost:3000

## Testing
cd backend
python -m pytest tests/ -v

## Environment Variables
Backend (.env):
- GEMINI_API_KEY=your_key
- SECRET_KEY=your_secret

Frontend (.env.local):
- NEXT_PUBLIC_API_URL=http://localhost:8000
